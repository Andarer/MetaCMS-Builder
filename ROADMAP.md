# Дорожная карта (Roadmap) — MetaCMS Builder 🗺️

Этот документ описывает стратегическое планирование развития платформы **MetaCMS Builder** с разделением по фазам разработки и статусам.

---

## 📈 Текущий статус выпусков: 
- **Основная версия стабильного ядра:** v1.3.0 (Released)
- **Активная бета-версия (Патч i18n & Темы):** v1.4.0 (In Progress)

---

## 🛠️ Матрица состояния функций (Feature State Matrix)

### 🟢 Released (Выпущено)
- **Интерактивный симулятор БД (Sandbox):** Возможность эмулировать работу и проверять валидность полей схемы таблиц локально без локального сервера.
- **Поддержка динамического перевода (MultiLang i18n):** Мгновенная локализация всего интерфейса на русский, английский и украинский языки.
- **Умные Световые Режимы:** Автоматическая смена тем (День, Ночь, Адаптивная, По расписанию).
- **Синхронизатор с ИИ Gemini:** Генерация кода Flutter и проектирование структуры колонок СУБД из одного текстового запроса.
- **Интеграция с GitHub-Pages:** Готовые Actions (.yml) для автоматического веб-деплоя.

### 🟡 In Progress / Testing (В разработке и тестировании)
- **Универсальные коннекторы Firebase (In Progress):** Автоматическая генерация схем Firestore и Storage-хранилищ.
- **Многопользовательское кэширование воркспейсов (Testing):** Шифрованное локальное хранилище для переключения учетных записей.
- **Сборка Android release-пакетов (Testing):** Тонкая отладка signingConfigs в Actions-процессах для публикации готовых APK/AAB-артефактов в ветках релизов.

### 🔵 Planned (Планируется)
- **Транслятор в NestJS/SQL (Planned):** Расширение поддерживаемых бэкендов генератора (генерация таблиц PostgreSQL и REST API эндпоинтов на Node.js).
- **Экспорт схем в GraphQL (Planned):** Автоматическое построение GraphQL-клиента и схем резолверов для фронтенда.
- **Динамический инспектор логов CI/CD (Planned):** Подключение вебсокетов для отображения реальных потоков сборки GitHub-конвейеров внутри мобильного приложения.

### 🔴 Deprecated (Устарело / Не рекомендуется к улучшению)
- **Локальный JSON-экспорт схем (Deprecated):** Будет удален в версии 2.0.0. Полностью заменен вложенными таблицами SQLite Room и ИИ-генерацией Dart-моделей.
- **Поддержка API Keys в SharedPreferences (Deprecated):** Небезопасно. Настоятельно рекомендуется инжектировать системные переменные через Gradle BuildConfig.
