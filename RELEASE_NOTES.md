# Заметки о выпуске (Release Notes) — MetaCMS Builder 📦

## Версия: v1.4.0-Beta (Сборка: Local Sandbox and MultiLang Engine)
**Дата выпуска:** 14 июня 2026 г.  
**Владелец релиза:** Open Source Maintainers  

---

### 📝 Обзор релиза

Выпуск версии **v1.4.0-Beta** направлен на глубокую консолидацию интернационализации, расширение режимов тем оформления и стабилизацию компиляции логики генерации схем. Все обновления полностью локализованы на русский язык и снабжены автоматической адаптацией на базе системного окружения Android.

---

### 🚀 Ключевые нововведения

#### 1. Многоязыковый Контент-Движок (i18n Sync)
- Введена поддержка Русского (основной по умолчанию), Английского и Украинского языков.
- Внедрена функция динамической транспиляции интерфейса в реальном времени;
- Интегрировано автоматическое определение языка локали пользователя (через `java.util.Locale.getDefault()`). При выборе "System" система автоматически определяет подходящую кодировку, падая на Русский в качестве базового fallback для нераспознанных локалей.

#### 2. Матрица Световых Режимов (Adaptive Theme Controller)
- **Day (Светлый режим):** Оптимизация контрастности для работы на открытом воздухе.
- **Night (Глубокий режим):** Защита зрения при слабом освещении на базе темно-серых Material 3 тонов.
- **System (Адаптивный):** Следование настройкам темы операционной системы устройства.
- **Scheduled (По расписанию):** Интеллектуальный расчет восхода и заката. С 6:00 до 20:00 включается дневной режим, с 20:00 до 5:59 — ночной автоматически.

#### 3. Архитектурная стабильность и Баг-фиксы
- Исправлена критическая проблема неоднозначности перегрузки (`Overload resolution ambiguity`) функции `getIconForModule` в модулях `DashboardScreen.kt` и `ModulesScreen.kt` путем изменения области видимости до `private`.

### 📂 Архивы и артефакты сборки

Конвейер автоматизации GitHub Actions автоматически генерирует и публикует следующие артефакты при установке нового тега релиза (например, `v1.4.0`):
1. **`app-debug.apk`** — Автономный тестировочный пакет Android-приложения (загружается из артефактов Github Runner).
2. **`metacms-web-pwa.zip`** — Архив статических веб-страниц Flutter PWA, готовый для развертывания под Nginx.
3. **`docker-compose.yml`** — Сценарий быстрого деплоя серверного контейнера в облаке или локально.

### 📥 Инструкция по обновлению и запуску

Для обновления вашей копии до последней версии выполните следующие команды:
```bash
# Получение изменений
$ git fetch origin
$ git checkout tags/v1.4.0-Beta

# Компиляция и сборка проекта локально
$ ./gradlew assembleDebug
```
